Program on Epiphany

Programming on Epiphany is similar to writing markdown. To add a program block, select either Javascript or Python when creating a new block.

The program code editor is also similar to the markdown editor. You can toggle the fullscreen mode if you like. The play button will not only save the change to the draft, but also execute the program. By default, only the execution result is displayed, but you can also choose to display source code alongside the result.

Epiphany provides a set of simple APIs you can use to make output, create an UI or request a webgl canvas. The same set of APIs is available to both the Javascript code and the Python code.

To generate text output, you can call the log function. Calling the clr() function will clear what's been printed so far.

log("This is a js block.")

You can also create a simple UI for each program block to interact with the program. This UI is based on dat.GUI. For example, the following code creates a simple button with a callback.

let run = function () {
    ...
}
let ui = createUi()
var ControlUI = function () {
    this.run = function () { run(); };
};
var text = new ControlUI();
ui.add(text, 'run');

For more examples on how to use the UI library, please visit dat.GUI.

Finally, you can request a canvas. Here is an example on how to request a canvas and setup a three.js context:

let canvas = getCanvas()
canvas.style = { width: 0, height: 0 }
const renderer = new THREE.WebGLRenderer({
	canvas: canvas,
	antialias: true,
	alpha: true
});

Write on Epiphany

Epiphany has a similar concept like Jupyter where an aritcle is formed by several blocks or cells. A cell can either be a piece of text or a program. In the future, Epiphany may support more block types than text and program, though they should fulfill most of your needs.

To create new content, navigate to the profile page once logged in, and click on the New Post button:

A blank post should be generated. With an empty Meta block you need to fill in. The first block of any article is always a Meta block. You can create it, delete it or move it. This block is generated for you automatically when creating a new post. A post must have one Meta block.

You need to finish this meta block by providing a title and optionally some tags. Once ready, click the Play button on the bottom right, this will generate a new post for you.

Now, a new post is created with a Meta block at the top. You may notice that if you move your mouse to a block, it will be highlighed with blue dash lines on the side, meanwhile, a few buttons will appear. These buttons are the edit button. You can use them to start the edit mode, remove a block, insert a block or reorder blocks.

Now, since we have no blocks in this empty post, the first step is pressing the Plus button to add a new block. Any newly added block has no type.

You need to provided the desired block type through the dropdown and then confirm by clicking Add.

We will cover how to program in Epiphany in the next session. For now, let's choose Markdown.

Once a Markdown block is created, a placeholder line is created. You can now write your post content. To do this, just click the Edit button.

This will switch the current block into the edit mode. In the edit mode, you will have a markdown editor as well as the rendered markdown result.

If you like a more focused writing experience, you can also switch to the fullscreen editing mode. In this mode, only the editer and the rendered result are displayed side by side. Once done, click on the Play button again, the block will be switched to the view mode again and the article is saved to the server.

A new post is by default a Draft until the post is published. In the Draft mode, only the author of the post could see it and make modification. To turn an article to the publish mode, click the Edit button of the Meta block again. Here you will see 2 buttons. The Publish/Unpublish button toggles the visibility of this article, whereas the Commit button save the current version to the published version.
